USPS Handbook AS-504, Space Requirements, is divided into several
separate Acrobat PDF® files. Keep the files in the folder created
during unzipping to ensure that links between them work.

The handbook's linked table of contents is in the file named
as504toc.pdf.

These files include an Acrobat index to enable fast searches.  In
the free Acrobat Reader with Search®, choose Edit>Search>Select
Indexes and add the index as504.pdx.  Then choose Edit>Search>Query
to conduct a very fast search of the entire Handbook AS-504. The 
Acrobat Reader with Search is available at Adobe Systems, Inc.'s 
Web site, http://www.adobe.com/products/acrobat/readstep2.html.
